-  node介绍
```
 node是一个开发平台
 javascript运行环境
 基于Chrome V8 JavaScript引擎构建
```
-  node优势
```
1.  完全是js语法
2.  超强的高并发能力
3.  高性能服务器
4.  开发周期短、开发成本低、学习成本低
```

- node特点
```
1. 事件驱动
2. 非阻塞I/O模型
3. 单线程
4. 开源生态系统npm
```

-  node能做什么
```
1. 提供API接口
2. 高性能I/O用于实时应用
3. 工具类应用
4. 等等等等
```
